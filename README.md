# Introduction
This **MERN** (MongoDB, Express, React, Node) stack project focuses on the **backend** side, especially **user authorization, [Docker](https://hub.docker.com/search?q=xinyuz0416), [K8s](https://github.com/XinyuZ0416/mern-ts-auth/tree/main/k8s),** and **[GitHub CI/CD](https://github.com/XinyuZ0416/mern-ts-auth/actions)**. 

It's written in **Typescript** using [Bcrypt](https://www.npmjs.com/package/bcrypt) package for password encryption and [express-session](https://www.npmjs.com/package/express-session) package for session management.
